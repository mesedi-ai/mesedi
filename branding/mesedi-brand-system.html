#FF8C42 is the load-bearing accent — same value as the dashboard --accent variable. Steel #7FA6B8 stays reserved for active/alert states in product UI.

Color rationale: the bronze-gradient version read warm but slightly muted next to the marketing landing's saturated orange. This palette retunes the gradient stops around the existing #FF8C42 accent so the mark and the landing accent are the same color family without a perceptual jump between hero text and logo.
Watch-out: use the gradient version only where you control the render pipeline. Slack, Notion, Twitter, and email-client favicon compressors will crush subtle gradient stops — ship the flat #FF8C42 variant to those surfaces.
